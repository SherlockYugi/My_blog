时隔近两个月的更新（

Author: Sherlockyugi Date: 四月 23, 2025  22:03:35

过了半个学期的突然更新（

主要原因是上个月发现了Github工作流出了bug，没法使两个库同步，经过一晚上琢磨仍然无解，便作罢。而今天是打完计设的第一天，彻底没活干了，一时兴起研究了下怎么修复，没想到按照[https://xulianjun.github.io/2024/09/08/Hexo + GitHub Action，零成本自动化部署个人博客/]把这个工作流文件复制下来就行了，明明我之前也是按照这个文件改的，为什么没用呢（

马上五一了，然而候补率百分之八十五的票竟仍在候补流程，明明我百分之六十的票两天就兑换成功了，岂可修。而回程的票还没想好坐哪班，看最后能给我剩哪班吧（

关于这次计设，算是我第一次作为策划带团队（是否带领存疑），总之深刻发现我确实没什么领导力，而且沟通能力也不行，导致一些功能和我最初设想小有差异。最主要的感觉还是与美术方面的沟通，不仅由于在整体项目上缺乏决断力，而且在细节上也沟通乏力，甚至很多除美术方面的部分如剧情都交给了美术，更感觉自己有很多的不足了（悲）。而作为补偿，做了挺多程序的活（自认为），但基本都是用AI生成脚本再微调数值，脚本很多部分也完全不懂，对未来的方向更加迷茫了（大悲）

基本上是走策划了，系统、关卡甚至是技术，感觉都有点偏好。但感觉未来游戏业像我一样的新人挺多的，有技术壁垒的岗位更好，但是技策招的又少，要求又高，感觉由此进入还是太难了（

还是先学会主业吧，半个学期没听课了，总不能挂了吧（（（
